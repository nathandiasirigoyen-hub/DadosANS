# DadosANS
Projeto comunitário que busca organizar e analisar os bancos de dados públicos que tratem de saúde suplementar no intuito de extrair insights e informações que possam subsidiar os beneficiários e gestores de planos de saúde brasileiros nas tomadas de decisão, especialmente aqueles que fazem parte de sistemas públicos ou de autogestão em saúde.
